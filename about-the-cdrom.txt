Appendix A
About the CD-ROM

The CD-ROM included with GNU/Linux Application Programming includes all 
example applications found in the book.


CD-ROM Folders

source: Contains all the code from examples in the book by chapter. 

figures: Contains all the figures in the book, by chapter.



OVERALL System Requirements

o	Linux with a 2.4 or 2.6 Kernel (tested with Fedora and Ubuntu)
o	Pentium I Processor or greater
o	CD-ROM drive
o	Hard drive
o	256 MBs of RAM
o	1 MB of hard drive space for the code examples.
